# 🚀 Alien Invasion

A simple 2D arcade game built with **Python** and **Pygame**, where you control a spaceship and defend Earth from waves of invading aliens. As you progress, the game becomes faster and more challenging, testing your reflexes and accuracy.

---

## 📖 What is this project?

This project is a recreation of the classic *Alien Invasion* arcade game using Python and the Pygame library.

The player controls a spaceship that can move left and right while shooting bullets to destroy incoming alien fleets. Each time an entire fleet is defeated, the next wave moves faster, increasing the difficulty. The game also keeps track of the player's score, level, remaining lives, and highest score.

The project was built to gain hands-on experience with object-oriented programming, game loops, collision detection, sprite management, and event handling in Pygame.

---

## ✨ Features

* Smooth spaceship movement
* Bullet shooting with firing limit
* Alien fleet generation
* Collision detection between bullets and aliens
* Increasing game difficulty after each level
* Score and high-score tracking
* Multiple player lives
* Play button to restart the game
* Clean object-oriented code structure

## 💡 Why is it interesting?

Although the game looks simple, it brings together many important programming concepts in one project.

It uses an object-oriented design where each game component (ship, aliens, bullets, scoreboard, settings, and buttons) is implemented as a separate class. The game also demonstrates collision detection using Pygame sprites, continuous game loops, event handling, dynamic difficulty scaling, and score management.

Building this project helped me understand how larger Python applications are organized and how multiple classes interact to create a complete game.
